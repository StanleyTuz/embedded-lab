If I were designing your curriculum today

I'd almost ignore formal courses for the first couple months.

Instead I'd use the Arduino as a laboratory and follow a sequence like:

Module 1: Digital I/O

Exercises:

Blink LED
Blink two LEDs at different rates
Button controls LED
Toggle LED on each button press
Module 2: State Machines

Exercises:

Traffic light
Multi-mode blinker
Menu system with one button
Module 3: Timing

Exercises:

Remove all delay()
Blink multiple LEDs independently
Periodic tasks
Module 4: Interrupts

Exercises:

Button interrupt
Timer interrupt
Compare polling vs interrupt implementations
Module 5: PWM

Exercises:

LED dimming
Servo control
Module 6: ADC

Exercises:

Potentiometer reading
Analog-controlled LED brightness
Module 7: UART

Exercises:

Command shell
Configuration commands
Module 8: I2C

Exercises:

OLED display
Sensor reading

This is actually very close to how many university labs are structured.
